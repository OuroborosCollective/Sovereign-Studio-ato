# Sovereign Runtime

## Brain gate

Every provider route must produce perception, analysis, plan, execution and learning layers. A response without execution patches is rejected before GitHub push.

## Launch readiness rail

Sovereign runtime imports the RepoP3N12 launch-readiness idea as deterministic TypeScript checks: safe repo inspection, task extraction, readiness evaluation, owner checklist, copywriting and workflow watch.

## Checks

- npm run type-check
- npm run lint
- npm run test:run
- npm run build:web
